Vue中的axios总结

发表于 2022-03-07

为什么要引入ajax

为了解决浏览器在每次向服务器请求数据的时候，服务器都需要返回整个界面的数据这一问题，产生了Ajax异步网络请求

  • Ajax(Asynchronous JavaScript and XML)：异步网络请求

  • Ajax能够让页面无刷新的请求数据

实现Ajax的方法

  • jQuery封装Ajax（有点小题大做）
  • 原生XMLHttpRequest（调用和配置比较繁琐）
  • Axios（最好）

Axios简介

并不是一种全新的技术，本质上是对原生XMLHttpRequest的封装，并且基于Promise以及符合全新的ES规范使其可以更便捷的进行Ajax请求

可用于nodejs的HTTP客户端和浏览器

smartschool问题总结

发表于 2022-03-06

本篇对上学期开发中出现的问题进行总结

1.重构时注意字段数据是否与接口一一对应

如图所示，当返回status的值非0或者1时，应该写一个错误返回。

2.提交问题

在接口需要传递参数为对象时，最好创建一个新的对象把数据传入其中，尽量不要直接使用data进行传参。

3.输入兜底

如图，记得写输入兜底，比如性别输入错误，填入一个非男或者非女的字符串，参数里就没有正确的sex字段提交了，所以加入检查并且写一个兜底返回。

4.async/await

async函数返回的是一个promise对象

await等待一个async函数的返回值也可以等任何表达式的结果

如果等到的不是Promise对象，则await的结果就是他等到的东西。如果是Promise对象，await将等对象resolve，将这个值作为结果。

5.this

每个新函数根据他是被如何调用的来定义这个函数的this值

  • 如果该函数是一个构造函数，this指针指向一个新的对象
  • 在严格模式下的函数调用下，this指向undefined
  • 如果函数是一个对象的方法，则它的this指向这个对象

箭头函数不会创建自己的this，它只会从自己的作用域链的上一层继承this
