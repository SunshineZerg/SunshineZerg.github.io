计算有序矩阵中的负数

发表于 2023-01-26

给你一个 m * n 的矩阵 grid，矩阵中的元素无论是按行还是按列，都以非递增顺序排列。 请你统计并返回 grid 中 负数 的数目。

提示：

m == grid.length
n == grid[i].length
1 <= m, n <= 100
-100 <= grid[i][j] <= 100

因为矩阵从左到右，从上到下都是降序，因此只要找到第一个负数或者0，他右边和下面的剩下所有数目都满足题目要求，转化题目为查找，采用二分查找的方法
