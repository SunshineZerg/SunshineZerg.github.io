腾讯云验证系统问题

在写用户换绑手机号功能时用到了腾讯云的验证系统，
在按 https://cloud.tencent.com/document/product/1110/36841 文档接入验证码后，点击触发按钮没有反应。
调试发现应该是 <script src="https://ssl.captcha.qq.com/TCaptcha.js"></script> 这一行代码引入文件失败，导致触发事件未被正确绑定。
src是es5写法，vue使用es6，网上找到解决方法为在export default中加入以下代码，直接使用DOM插入js文件：
mounted() { let script = document.createElement('script'); script.type = 'text/javascript'; script.src = 'https://ssl.captcha.qq.com/TCaptcha.js'; document.body.appendChild(script); }
另外还有使用render的另一种解决方法，但vue打包时报错：
export default { components: { 'xxx-js': { render(createElement) { return createElement( 'script', { attrs: { type: 'text/javascript', src: '你的js文件地址', }, }, ); }, }, }, }
