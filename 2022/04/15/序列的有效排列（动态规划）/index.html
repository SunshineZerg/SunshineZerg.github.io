序列的有效排列（动态规划）

发表于 2022-04-15

给定一个长度为 n 的字符串 s ，其中 s[i] 是:

“D” 意味着减少，或者
“I” 意味着增加
有效排列 是对有 n + 1 个在 [0, n] 范围内的整数的一个排列 perm ，使得对所有的 i：

如果 s[i] == ‘D’，那么 perm[i] > perm[i+1]，以及；
如果 s[i] == ‘I’，那么 perm[i] < perm[i+1]。

示例 1：

输入：s = “DID”
输出：5
解释：
(0, 1, 2, 3) 的五个有效排列是：
(1, 0, 3, 2)
(2, 0, 3, 1)
(2, 1, 3, 0)
(3, 0, 2, 1)
(3, 1, 2, 0)

S[i-1] == ‘I’, 尾数为j的所有排列情况就是在i-1中所有尾数范围在[0, j)(小于j)的所有排列情况的累加
