K次串联后的最大子数组之和

给定一个整数数组 arr 和一个整数 k ，通过重复 k 次来修改数组。

例如，如果 arr = [1, 2] ， k = 3 ，那么修改后的数组将是 [1, 2, 1, 2, 1, 2] 。

返回修改后的数组中的最大的子数组之和。注意，子数组长度可以是 0，在这种情况下它的总和也是 0。

由于 结果可能会很大，需要返回的 109 + 7 的 模 。

示例 1：

输入：arr = [1,2], k = 3
输出：9
示例 2：

输入：arr = [1,-2,1], k = 5
输出：2
示例 3：

输入：arr = [-1,-2], k = 7
输出：0

提示：

1 <= arr.length <= 105
1 <= k <= 105
-104 <= arr[i] <= 104

  • k=0 返回0
  • k=1 题目转化为求数组最大子串和
  • k>=2 分数组和大于0小于0两种情况

如果大于0

要取到最大值相当于第一个取最大后缀和最后一个取最大前缀和，中间每个数组都贡献一个sum

此时

max = postmax+sum*(k-2)+premax;

如果小于0

最大值是0，单个数组的最大子串和，最大前缀和和最大后缀和的和三者的最大值

此时

max = Math.max(postmax+premax,maxmax,0);
